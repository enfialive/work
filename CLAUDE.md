# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 工作原则

1. **使用中文对话** — 所有交互、回复、代码注释和文档编写均使用中文，除非代码本身（变量名、函数名等）需要使用英文。
2. **尽量减少幻觉** — 遇到不确定或无法验证的信息时，明确说明不确定性，不做无根据的推测。优先查阅仓库内已有文件和代码后再做判断，不凭空编造内容。

## 仓库概述

这不是传统软件项目，而是个人工作空间，包含：
- 独立的单文件 HTML 工具（无需构建，浏览器直接打开）
- 专利 OA 答复文档及相关资料
- 抗体专利申请撰写指导文件

没有构建系统、包管理器、测试框架或 CI/CD 配置。

## 项目文件说明

### HTML 工具

- **`cdr-tool.html`** — 抗体可变区 CDR 标注工具。在浏览器中打开，输入氨基酸序列，选择编号系统（IMGT/Kabat/Chothia/Martin）和标注方案，输出 CDR/FR 区域标注结果。纯前端，无外部依赖。
- **`projectexpense.html`** — 项目支出管理工具。带侧边栏的多项目管理界面，支持按月份记录和查看支出。纯前端，使用 localStorage 存储数据。
- **`calendar-todo.html`** — 日历待办工具。以日历视图管理待办事项，支持周视图切换。纯前端，使用 localStorage 存储数据。
- **`timeline-tool.html`** — 药物研发全景时间线工具。双轨时间线展示药物研发关键节点。纯前端，无外部依赖。

## 开发注意事项

- HTML 工具为纯静态单文件，修改后直接在浏览器打开测试，无需启动服务器
- 所有数据存储在浏览器 localStorage 中，无后端
- `.gitignore` 已排除敏感/非公开文件（申请文件 .docx、大型参考文件 .pptx/.zip、临时下载等）
- 中文字符编码使用 UTF-8
